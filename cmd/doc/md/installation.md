### Installation

There are multiple ways of obtaining ez-cgi:

##### Brew
```bash
brew tap raphaelreyna/homebrew-repo
brew install ez-cgi
```

##### Go get
```bash
go get -u -v github.com/raphaelreyna/ez-cgi/cmd
```

##### Compiling from source
```bash
git clone github.com/raphaelreyna/ez-cgi
cd ez-cgi/cmd
sudo make install
```
